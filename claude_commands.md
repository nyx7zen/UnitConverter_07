# Claude Code 기본 명령어 참조

## 대화 관련

| 명령어 | 기능 |
|---|---|
| `/help` | 사용 가능한 모든 명령어 목록 보기 |
| `/clear` | 현재 대화 내용 초기화 (새 대화 시작) |
| `/compact` | 대화 내용 압축 (context 절약) |
| `/context` | 현재 context 사용량 확인 |

## 모델 관련

| 명령어 | 기능 |
|---|---|
| `/model` | 현재 사용 중인 모델 확인 및 변경 |
| `/config` | 설정 확인 및 변경 (테마, 모델 등) |

## 작업 관련

| 명령어 | 기능 |
|---|---|
| `/init` | 현재 프로젝트에 CLAUDE.md 자동 생성 |
| `/review` | 현재 브랜치 코드 리뷰 |
| `/status` | 현재 상태 확인 |

## 실행 취소

| 명령어 | 기능 |
|---|---|
| `Esc` 또는 `Ctrl+C` | 현재 실행 중인 작업 중단 |

## 이 프로젝트에서 자주 쓸 명령어

```
/clear     → 새 단계 시작 전 대화 초기화
/compact   → 대화가 길어졌을 때 압축
/context   → context 여유 확인
```

> **팁:** 대화창이 길어지면 `/compact` 후 계속 작업하면 됩니다.
> CLAUDE.md가 있으면 압축 후에도 프로젝트 규칙은 유지됩니다.
