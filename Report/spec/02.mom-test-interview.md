# Mom Test — 학습자·사용자 인터뷰

## 질문 10개 (요약)

1. 마지막 단위 오류 사건 — 과거 경험·수정 행동·시간 리스크
2. 자릿수 허용 기준 — 실제 판단·오차 비용
3. 포맷 불일치 피드백 — CSV/JSON/표 협업 시간
4. 비율 변경 — 설정 vs 코드·승인·회귀
5. 신규 단위 ad-hoc 추가 — 첫 깨짐
6. 검증 없이 잘못된 출력 — 검증 강화 전후
7. 실습 시간 분포 — 분석/구현/테스트
8. 테스트 선행 vs 후행 — 잡은 버그·리팩터
9. AI 산출물 직접 검증 — 최근 실패 사례
10. 한 파일 vs 분리 — 유지·인수인계

## 진짜 위험 가정

- 과제 완료 우선, 테스트·설정 스킵
