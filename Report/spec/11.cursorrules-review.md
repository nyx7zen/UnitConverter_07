# `.cursorrules` 검토 보고

## 1) YAML 문법

- 빈 최상위 키 null 파싱 — 통상 유효
- `tdd_rules` 중첩·`≥` 비ASCII — UTF-8 환경 OK

## 2) 누락 섹션

- 8키 존재, **내용은 `tdd_rules`만** 채움
- `ai_behavior` 비어 있음

## 3) tdd_rules vs forbidden

- `forbidden` 비어 **직접 충돌 없음**
- green 하드코딩 vs 향후 forbidden literal 금지 **잠재 충돌**

## 4) ai_behavior

- 규칙 없음 → **평가 불가**
- tdd_rules 내 커버리지·커밋 전 테스트·설계 문서 참조는 **AI 자동 준수 어려움**
